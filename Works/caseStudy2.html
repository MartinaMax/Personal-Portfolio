Responsive webpage

Year:

2022

Tasks:

Design, Coding

Tech:

HTML, CSS

GitHub repository

The challenge was to realize single responsive webpage from a given mock up, using HTML & CSS. As well as the mock up we were given the font, colours, icons and images. The mobile first approach was required for realization of this project as the mock up was a mobile version of the webpage.

I started with creation of design for all the remaining screens...Tablet and Desktop. Adjusting the layout; picture, heading and body text size.

Then I visualized it with HTML and CSS with use of Media Queries.
